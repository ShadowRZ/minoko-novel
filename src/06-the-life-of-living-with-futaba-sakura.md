<p style="text-indent: 0"><b>-&gt; 2XXX-XX-XX+1 15:30</b>
<br/>
“愿望实现的感觉怎么样，Minoko？”

“好极了，双叶！”

已经回到现实的 Minoko 活蹦乱跳的，像个没有长大的孩子。

“Minoko 别激动过度啊”，双叶心里这么想着？

这时有一个熟悉的声音传来：“双叶，那个人真的没问题吗？”

是摩尔加纳的声音。声音很小，只有佐仓双叶能够听到。于是双叶看向那只猫——摩尔加纳在现实里是一只猫——并且表示：“像个小孩子一样也没什么问题啊，而且他看起来很天真的，应该不会对我们构成什么威胁。”

摩尔加纳无语了：“真的要让我们怪盗团相信这个普通人？他又不是三岛由辉。”

佐仓双叶只好表示：“他喜欢我的存在，而且我感觉他心事很多的。”

摩尔加纳被吓了一跳：“他他他……他喜欢你？”

“嗯，我看出来了。”

<br/>

此时已经冷静下来的 Minoko 对佐仓双叶说：“我想和你聊一些事情，可以吗？”

佐仓双叶同意了，而摩尔加纳这时想着「这个人会和双叶说什么呢？」也跟上去听讲话了。

<br/>

Minoko 和佐仓双叶坐在沙发上。

“所以说 Minoko，你想谈什么呢？”

“嗯……”

摩尔加纳此时也在静听 Minoko 会说出的话语。

“双叶你为什么这么快就接受了自己从 P5 的世界里来我们这边的事情？”

“也许……是因为自己曾经有过去别的作品的经历。”

摩尔加纳也想起来那些去别的作品里联动的经历。

“那，这次你是怎么来到这里的？”

双叶不语，随后只扔出一句：“我不知道。”

摩尔加纳大惊：“你，你竟然不知道？”

Minoko 被吓了一小下，不过很快又恢复平静。

双叶对摩尔加纳表示：“嗯，真的，因为我其实……并不是从印象空间被带入这里的，而是在我老家。”

这是摩尔加纳从未想到的事情。在它的印象里，怪盗团都是通过印象空间进入其他作品的世界的。

双叶又补充：“我感觉自己周围的世界突然变成了这里的样子，而旁边就是这位 Minoko。”说着又指向 Minoko。

Minoko 对这个场面有点熟悉。

“真的是和 Re:CREATORS 里一样的展开，我感觉……你是被某个人拉入我们这边的世界的。”

那么……是谁呢？

谁把双叶拉入这个世界的？

这时摩尔加纳接上一句话：

“吾辈在想一个问题，会不会结城理和白钟直斗也和你预想的一样，也是被某个人拉入这个世界的？”

Minoko 沉默了一番。

“Emmm……要不我再细问一下？”

双叶赶紧接上：“不用了，不用了，别老麻烦他们。”

<br/>

<p style="text-indent: 0"><b>-&gt; 2XXX-XX-XX+1 16:00</b>
<br/>
<br/>

“那个，你们不试一下在这里能不能召唤人格面具的吗？”

Minoko 想到了这件事。

<br/>

一，二，三……

“Persona！”

但是实际上只有一声。另一声并没有出现的原因是佐仓双叶觉得 Minoko 家里地方太小，不好召唤她的人格面具。不过她发现，在她有了想要召唤人格面具的想法后，她的着装立刻就变了。

摩尔加纳则更是惊讶。因为它居然在刚想召唤人格面具的一瞬间就不再是一只猫的样子了。

“……吾辈觉得这太不可思议了……NAVI 你的着装是怎么回事？”

“我就是想了一下……”

过了几分钟，双叶的着装没有变成平常的样子，摩尔加纳也并未变回一只猫。

“难道我们还要主动变回去？”

“也许是吧，你们应该是得主动切换形态。”

果然，正如 Minoko 所想，他们想了一下之后就变回了原来的样子。

“真是神奇啊。不过……”

摩尔加纳说话时突然停顿一下。

“……Minoko 你是怎么想到这些的？”

因为 Minoko 在说这些话的时候，就仿佛知道这些事情一样。

“嗯……我猜的。”

“你，你怎么能猜的这么准的？”

“我对每个设定都有着自己的理解，这只是我的其中一个想法，没想到一次成功了呢。”

佐仓双叶这时也附和道：“喔喔，没想到你会这么厉害，居然有这么多的想法，也许有一天你真的可以写出一部非常棒的小说呢！”

“真的吗，双叶？”

“嗯，Minoko！

“顺便一提，我也想趁着这个机会多了解你呢！”

然而 Minoko 这时却卸下了高兴的表情，表现出一种莫名的悲伤。

佐仓双叶和摩尔加纳都有些疑惑不解。

“其实，我……”

Minoko 说出这句话时更显悲伤。

佐仓双叶急忙上前开导 Minoko：“那个，既然你这么喜欢我，那就让我做你的知心朋友，好吗？想说什么都可以对我说。”

<br/>

Minoko 平静下来了。

“不过，双叶，我们能去印象空间谈吗？”

……摩尔加纳有些无语，但佐仓双叶却立刻做出了回应。

“可以呀！摩尔加纳要不要也过来？”

摩尔加纳立刻回应道：“这就不用了，吾辈可不想偷听 Minoko 的话，我就在这里看着 Minoko 家，以防万一。”

“好的。”

“那，走吧，Minoko。”

<br/>

<p style="text-indent: 0"><b>-&gt; 2XXX-XX-XX+1 16:05 印象空间</b>
<br/>
<br/>

Minoko 用了五分钟时间把他想说的话告诉了双叶。

<br/>

Minoko 的高中生活原本是在一个声望极好的公立学校开始的。

只是可惜，尽管他有着极好的成绩（这也是他能够来到这所学校的原因），但是他……怎么说呢，其实很有攻击性。虽然首次见到双叶是他有提到这一点，但他没细说他有什么样的攻击性。展开说的话有两点：一是顶嘴，二是威胁。更令人抓狂的是，他还会在此之后歇斯底里的，觉得大家都害他。

Minoko 第一次休学之后做了手术，这期间他和 Arch Linux 中文社区的人聊天的事情被父母发现后，被迫做了一次“断舍离”，但仅仅是为了迅速逃离这处境。至年末又回归群里。次年在另一班级重新上课，但又被遣退，原因和上次一样。后来他上了一所私立高中，是寄宿制的。这才上完了高二。这里的人们比较包容他的这种样子，但他还是会闹情绪。

<br/>

……

佐仓双叶已经不知道该说什么好了。

过了许久，她才说出一句话：“某天我带你看心理医生吧。”

Minoko 却提出了另外一个请求：“不，我希望……你能偷走我的心。”

佐仓双叶有些疑惑：“我，我真的应该……偷走你的心？”

Minoko 对此十分坚定。

<br/>

“喂，Minoko 家门铃响了！”

摩尔加纳的声音打断了二人。

“Minoko 你要不要去回应一下？”

听到双叶这句话的 Minoko 立刻就从印象空间出来回到他的家，回应了门铃。摩尔加纳也偷偷跟了出来。

<br/>

<p style="text-indent: 0"><b>-&gt; 2XXX-XX-XX+1 16:07</b>
<br/>
<br/>

来者是 Minoko 的父母。

<br/>

摩尔加纳在这个过程中，并没有听到争吵和咬牙切齿和活蹦乱跳，摇头晃脑。

看起来很正常的样子。

<br/>

还在印象空间的佐仓双叶听到摩尔加纳回来做的报告后，感觉 Minoko 已经和当初他自己说的那种样子不一样了，那种「不正常」的样子。

所以 Minoko 现在唯一的问题已经只剩他的内心了，是个和他们的领域很贴近的问题呢。

“感觉这些天 Minoko 改变很多呢。”

这样想着的双叶又和摩尔加纳回到了现实世界，然后坐到沙发上。

双叶把耳机连接到手机上并且打开了异世界导航。

接下来，她在目标主体的名字里输入 Minoko。

结果是异世界导航起了反应，提示已经找到目标。

“那……”

摩尔加纳有点不知所措。

“MONA，这个……以后再说吧。咱们先看看 Minoko 的状态吧。”

“……行，NAVI。”

<br/>

佐仓双叶正式向 Minoko 提出了全天候观测他的请求，Minoko 不但同意了，而且借用了双叶的电脑编写了监控自己的程序。

然而摩尔加纳的观测请求却被拒绝了：“对不起啊摩尔加纳，这里不允许存在宠物的说……”

“宠物……”

摩尔加纳估计无法接受吧。

“……那吾辈就退回印象空间了。”

向摩尔加纳报告观测数据的工作就这么无声地由佐仓双叶承担了——很简单，她什么都不会忘记。虽然过去这种能力给她带来了麻烦，但是现在这个能力可以让她不需要把观测数据记录到手机上（尽管双叶仍旧留存观测数据），直接口述给摩尔加纳。

<br/>

就这样，Minoko 被佐仓双叶观测的微妙的生活正式开始了。

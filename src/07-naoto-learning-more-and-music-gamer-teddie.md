<p style="text-indent: 0"><b>-&gt; 2XXX-XX-XX+1 15:30</b>

<br/>

又回来了呢。

这个对叶琴玉来说最熟悉的地方。

<br/>

从印象空间回来的叶琴玉和白钟直斗现在正在叶琴玉的家里。

叶琴玉还在回味着之前在印象空间里的事情。

白钟直斗坐在一边，直截了当地问萌狼：“你是怎么知道佐仓双叶的事情的？还有，你和 Minoko 到底什么关系？以及……你怎么知道这个叫什么，印象空间的事情？”

叶琴玉同意回答这个问题，但她表示：“我只能用文字叙述我和 Minoko 的关系。”

<br/>

Minoko 很早就在 #archlinux-cn 了，然后在两年前他和叶琴玉相遇了。

具体是怎么碰到 Minoko 的她自己其实也忘了。

不过后来的事情她也明白，Minoko 彻底把叶琴玉当做了他的知心朋友，而叶琴玉呢？也渐渐地对 Minoko 有点珍惜。她在 Minoko 的驱使下把 P5，P5R 和 P5S 都玩通关了，她还给 Minoko 直播 PDFT，她和 Minoko 甚至还发展了共同的爱好，至于是什么她也没说。

当初 Minoko 被迫离开 #archlinux-cn 时她也感到很伤心，但是没有办法，她也做不了什么。因为在 Minoko 的父母眼里，也许她不能被称为朋友。

除了这些，叶琴玉也把自己了解的一些 Minoko 的现状告诉了白钟直斗，这样就算回答了第一个问题。

但对于第二个问题她没说一句话，而是径直走到她的 PS4 前。

白钟直斗也跟了上去。

然后她把 PS4 的电源打开，启动游戏。她的想法就是借此介绍双叶那边的事情以及让白钟直斗理解一些事情。

白钟直斗有点惊讶：“所以佐仓双叶是这个游戏里的角色？”

叶琴玉表示：“嗯，难道佐仓双叶没和你说吗？”

“啊，对啊。不过我还是有些不能相信，我和她竟不过只是一介游戏里的角色而已，我们明明过了那么真实的生活……”

“白钟直斗，我其实……可以理解你的心情。”

白钟直斗已经感到不可思议了。

“因为我其实在别的游戏里看到了和你一样的心情。”

至于是什么游戏，因为各种游戏本身的原因她并没有说。

总之，叶琴玉借着 P5R 里的 My Palace 这一鉴赏功能向白钟直斗从头到尾介绍了一番心之怪盗团的经历。

<br/>

白钟直斗此时也有了自己的想法：

“这个佐仓双叶和我们好像呢，她也是在自己的内心世界里碰上了另一个自己，然后那个自己化为了人格面具呢。

“不过话说回来，当初我和心之怪盗团相遇的时候，我还以为他们的行动彬彬有礼，结果他们……他们是强盗团吧？一定是心之强盗团吧？？”

心之强盗团……

“白钟直斗，你可别这么贬低他们，还有……”

叶琴玉低声对白钟直斗说：“千万不要对他们说他们是心之强盗团，他们会不开心的。”

“……嗯。”

“还有，叶琴玉，怪盗团的人怎么个个都很惨？你这游戏叫《女神异闻录5》，我觉得这应该叫「比惨异闻录5」吧。”

比惨异闻录……

叶琴玉已经无力吐槽了。

所以说白钟直斗的关注点怎么会这么奇怪啊！

<br/>

<p style="text-indent: 0"><b>-&gt; 2XXX-XX-XX+1 16:00</b>

<br/>

“心之怪盗团真有意思，不但到处探索别人的内心，甚至还和 AI 少女与警察交手，然后警察也觉醒了……”

这是白钟直斗对叶琴玉谈到的怪盗团在 P5S 里的经历的想法。

“真是，比我想象的还厉害。我说……”

叶琴玉很好奇白钟直斗要说什么。

“……我可以成为心之怪盗吗？”

什么？

她，她要成为心之怪盗？

叶琴玉有些不知所措，吞吞吐吐地回答道：“这个……我不知道，我看看吧。”

白钟直斗略有失望。

<br/>

这时只听一声巨响。

好像是从客厅里传出来的。

二人前去看时，发现有只布偶趴在地上。

是 Kuma！他来这里干什么？

等到二人问 Kuma 的时候，他的回应是：“我觉得电视里太无聊，所以从电视里出来看看你们 Kuma。”

当 Kuma 看向叶琴玉时，Kuma 做出了一个惊人的举动：这个布偶装上的拉链突然自己拉开，当完全拉开后，头套掉落。

里面居然有个人！

叶琴玉被里面的纯美少年迷住了。

<br/>

“所以……嗯……Kuma 你……？”

“我都说了要来看看你们的世界 Kuma。”

<br/>

然而叶琴玉也没想过应该怎么给 Kuma 介绍这里的世界——因为这就是她自然地生活的世界，她已经想不出什么特别的不同了。

于是叶琴玉先把 Kuma 带到了她日常光顾的华卡音舞店，然后又打了一遍音游。

Kuma 被这从未见过的场景看得入迷，并提出自己也想打一场。

叶琴玉再度不知所措。她并不想把她自己的号借给 Kuma 玩，但是怎么办呢？Kuma 看起来也不像是有手机的样子。

白钟直斗提出了去找店长的提议。

只是，因为这个华卡音舞店采用的是全自动无人看管的系统，所以是否还会有店长也是个未知数。

这时三人看到一位工作人员。

“你好，我想问一下这里有没有店长？”

“嗯，有的，这边请。”

真的有店长啊！

<br/>

等到三人来到店长面前时，店长居然马上认出了白钟直斗和 Kuma。

“你们……我好像在玩游戏的时候看到你们了。”

这句话让 Kuma 和白钟直斗感到有些不太自在。

“……我不认识他而他居然认识我……”

“所以店长怎么会认识我？”

店长的话却让叶琴玉大吃一惊。

<br/>

这个店长，在开这家华卡音舞店前，是一个游戏主播，而且不是一般的主播，是和直播平台签订了合同，任期内永远只能是是游戏主播的主播。这期间他直播了有二十几个游戏吧，其中就包括 P4G，这是他直播生涯中最令他印象深刻的游戏。

“那，这个世界上是不是还有许多认识我的人？”

白钟直斗已经更不自在了。

“我想是的。

“然后呢，两年前我的合同到期了，于是我的游戏生涯差不多算是结束了。这时听说华卡音舞很火的于是就这么开了个店。这期间看其他人玩的时候我也有点蠢蠢欲动，于是也去开了个帐号玩了几局。结果呢，”

店长微微一笑。

“战绩特别差呢，果然我还是适合去打游戏呢。”

这时店长好像想起了什么。

“对了，你们找我有什么事？”

叶琴玉说明了来意，表示 Kuma 想玩华卡音舞。

“这样啊……那，我把我的帐号给 Kuma 玩了。”

什……

“反正我也不会玩音游了。让 Kuma 继承我的音游梦想吧。”

<br/>

<p style="text-indent: 0"><b>-&gt; 2XXX-XX-XX+1 16:45</b>

<br/>

Kuma 就这么拿着店长的帐号玩了起来。

叶琴玉和白钟直斗观战后发现，Kuma 的玩法不逊色于叶琴玉，很快就达到了和叶琴玉同样的水平。

同步率 98%。

真的是，令人万万没有想到呢。

<br/>

“这个游戏还很好玩的呢，我下次也要来呢。”

Kuma 像是打开了新世界的大门一样。

而叶琴玉呢？正在和店长商量要不要再用一部手机给 Kuma 开个新的帐号。

商量的结果是：要，名字就叫熊田，原因很简单——八十稻羽镇的居民就这么称呼他的。

<br/>

现在叶琴玉又要带一个音游人（？）了。
